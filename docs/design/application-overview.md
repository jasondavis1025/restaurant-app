# Restaurant Application Overview Goals

## Purpose

Online ordering platform for restaurant supporting customers, employees, and administrators.

## Applications

- Customer Web App
- Staff Web App
- Backend API

## Customer Features

- Browse menu
- Customize dishes
- Guest checkout
- Account registration
- Save favorite orders
- Order history
- Order tracking

## Staff Features

- View incoming orders
- Update order status
- Manage menu availability

## Admin Features

- Manage employees
- Manage locations
- Manage menu
- View analytics

## Technology Stack

- Angular
- Express
- PostgreSQL
- TypeScript
