About Page

Demo Restaurant is a demo restaurant website built to showcase a modern online ordering experience.

This project includes menu browsing, customer accounts, customizable orders, online ordering, and support for multiple restaurant locations.

Frontend

I built this site with Angular, Node.js, Express, Typescript, and PostgreSQL.

I'm familiar with many versions of Angular. For this project, I use version 22 of Angular, which is the latest version at the time of development.

This project fully utilizes a Zoneless infrastructure, with full reliance on Signals.

For the design, I fully coded this site with SCSS, instead of taking a pre-built approach with tools like Tailwind or Bootstrap.

Backend

The backend is built with Node.js and Express, and it uses PostgreSQL as the database.

No ORM's where used. I built the database queries with raw SQL to demonstrate capability as a database engineer.

UI/UX Design & Db Modeling

In addition to coding, I am a professional UI/UX designer.

The database was coded on DbDiagram.io and the ERD can be found in the project repository.
